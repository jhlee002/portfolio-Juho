문제 상황

배달 진행 과정에서 DeliveryStatus(라이더)와 OrderStatus(주문고객/점주)가 호환되지 않아 배달 단계가 연동되지 않는 문제가 발생했습니다.

원인

배달 상태를 enum 값으로 관리하고 있었지만, front와 back에서 사용하는 상태 값이 서로 다르게 설정되어 있었습니다.

해결 방법
  • DeliveryStatus와 OrderStatus에서 사용할 enum 값을 통일
  • front와 back의 상태 값을 동일하게 수정
  • back-end에서 명령 기반 방식을 적용
  • 하나의 메서드로 두 가지 상태 전이가 함께 이루어지도록 수정
  • DeliveryStatus와 OrderStatus가 병행 진행되도록 개선
